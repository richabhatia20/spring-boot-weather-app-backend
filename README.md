# spring-boot-weather-app-backend
This repo contains Spring Boot backend code for weather-react-app 


# Dependencies:
* Spring boot data jpa
* Spring boot starter web
* H2 database
* Spring boot starter test

# Dev Environment
* Intelli J
* Tomcat 8.5
* Gradle
* Sprint Boot 1.5.7
* Java 1.8
